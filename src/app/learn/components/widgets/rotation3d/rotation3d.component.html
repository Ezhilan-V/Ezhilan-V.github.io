Three ways to describe the same orientation. Drag pitch toward ±90° and watch the gimbal-lock preset: yaw and roll start affecting the same physical motion. Quaternions don't have this problem - every unit quaternion maps to exactly one orientation, and slerp interpolates smoothly even through poles.
