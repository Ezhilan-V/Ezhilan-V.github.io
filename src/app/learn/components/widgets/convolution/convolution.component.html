Each output pixel is a weighted sum of the 3×3 neighbourhood: out = Σᵢ kᵢ · srcᵢ. For positive-sum kernels (blur, Gaussian) we normalise by the sum. For edge kernels (Sobel, Laplacian) we add 128 so that flat regions show as mid-grey and edges appear as deviations. Tweak any cell by hand to see what it does.
